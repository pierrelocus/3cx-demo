3CX CRM Template full integration

3CX CRM Template full integration lets you generate the CRM Template XML File for 3CX and implements all capabilities of 3CX CRM Template (Contact Search/Create, Call Logs, Livechat Logs)

Compatible with Odoo SH and On Premise versions


Configuration steps

  • 1/ Install the module.
  • 2/ Inside Odoo Contacts, click on the new menu "3CX Configuration", Generate the config file, and download it.
  • 3/ Inside 3CX Admin -> Settings -> CRM, click "+ Add" to upload the downloaded Template and activate the functionalities you want.
  • 4/ To automatically open the Contact Page in Odoo when receiving a call, go in your 3CX WebClient -> Settings -> Integration and tick the box "Automatically open contact URL when CRM integration is enabled".
  • You're all set!
  • Note: Call and Livechat Logs will be added to the contact's Chatter
  • Note: You must be connected to your Odoo to open the Contact Page. Else you will be redirected to the Login page of your Odoo.
